### sbus値の読み取り

sbusデータを
throttleは 0 ~ 100
roll, pitch, yawは -100 ~ 100
スイッチは 0, 1, 2
の範囲に変換している

### センサーデータの読み取り

float accel[3]; float gyro[3];を取得
imu_ekfで float angle[3]に変換

### PID制御の実装

sbus値を目標値に変換
sbus値（-100 ~ 100）を目標角度に変換(例: -100 -> -30度, 0 -> 0度, 100 -> 30度)

目標角度と現在の角度の差を計算して目標角速度を求める

目標角速度と現在の角速度の差を計算してPID制御を行う
（現在の角速度はセンサーから読み取った生データ）

### モーター出力の計算
PID制御の出力をモーター出力に変換
pitch roll yawのPID結果とthrottle（手動: 0~100、自動: 高度pid結果）を組み合わせてモーター出力を計算
